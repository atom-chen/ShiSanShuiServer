命名规则：
    类成员变量 以 m_ 开头  后跟类型说明符 
            m_nXXX 表示数字类型数据(number)
            m_bXXX 表示boolean 类型数据(true or false)
            m_strXXX 表示字符串类型 (string)
            m_stXXX 表示结构类型数据 
            m_tbXXX  表示table 类型数据(table)
            m_arrXXX 表示为数组类型的table
